# Legacy Planning Archive

This folder contains harvested strategy notes and prototype artifacts that are **not** current roadmap documents.

Use the top-level `idea/` docs for current planning:

- `controlkeel-product-strategy-plan.md` is the current strategy document.
- `controlkeel-final-build-plan.md` is the canonical remaining-work document.

Everything under `idea/archive/` is historical source material:

- legacy framing and product-thesis notes
- retired prototype artifacts
- old branding, packaging, or market-research explorations

These files are still useful for provenance and context, but they should not be treated as the source of truth for shipped capability, roadmap status, or active implementation scope.

The original raw ideation prompt that seeded this folder has been intentionally removed from the active planning surface. Its intent now lives in the current strategy doc and these archived legacy notes.
